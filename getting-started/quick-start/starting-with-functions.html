🔌 Introdução com funções

Visão geral

Você sabia que o webui aberto pode se conectar a quasequalquer coisa-Não apenas APIs compatíveis com o OpenAI?

Obrigado aFunções de tubo, você pode trazer serviços que não suportam a API do OpenAI (comoAntrópico, Assistente de casa, pesquisa do Google, ouqualquerPython CodeBase). Sem restrições nos modelos LLMS ou IA: se você pode automatizá -lo no Python, poderá transformá -lo em um plug -in para o Open Webui!

Este guia orienta você a configurar sua primeira função de tubo, usando oTubo antrópicoplugin como exemplo.

O que são funções de tubulação?

As funções de tubo são plugins “trazer-seu-mesmo modelo (ou ferramenta)”:

  • Aja como modelos: Eles aparecem como modelos selecionáveis ​​na sua barra lateral aberta da Webui.
  • Flexível: Integrar -se aqualquerBack -end, API ou fluxo de trabalho - não é necessária compatibilidade do OpenAI.
  • Não é necessário LLM: Você pode criar plugins para pesquisa, automação residencial, clima, bancos de dados ou o que quiser.
  • Python puro: Toda a lógica é o código python que é executadodiretamente dentroSeu webui (então tenha cuidado com o que você habilita!).

Etapa 1: encontre uma função de tubo para tentar

Vamos integrarAntrópicoCom o Open Webui-mesmo que o Anthropic suporta apenas sua própria API nativa (não terminais compatíveis com o OpenAI)!

  1. Vá para oPágina de função de bate -papo antrópica

  2. CliquePegar

Etapa 2: importe a função para abrir Webui

Um modal aparecerá:

  1. Digite o seu URL do Open Webui (por exemplo,,http://localhost:3000) no prompt.
  2. CliqueImportar para abrir webui

Você será redirecionadodiretamente para o editor de funçõesdentro da sua instância de corrida do Open Webui.

Etapa 3: revisão e salvar

  • Você verá todo o código Python e configuração da função do tubo.
  • Importante:Funções executam python arbitrário! Revise o Código de Segurança e instale apenas a partir de fontes em que você confia.
  • Se você está satisfeito com o código, cliqueSalvarPara adicioná -lo à sua instância.

Etapa 4: Ative a função

Sua nova função de tubo está agora disponível, masdeve estar ativado

  1. Alterne o toggler para ativar a função.

Etapa 5: insira todas as teclas de API necessárias via válvulas

Algumas funções precisam de credenciais (como a chave da API do Anthropic):

  1. Clique no ícone de engrenagem ao lado do interruptor para abrir oVálvulasconfiguração.

  2. Insira a (s) chave (s) da API necessária para a função do tubo.

Etapa 6: comece a usar seu novo plugin!

  • A nova função agora aparece como um "modelo" selecionável na interface de bate -papo.
  • SelecioneAnthropic(ou o que você instalou) e comece a conversar!

🎉 É isso - você está conectado a qualquer coisa!

  • Funções de tubulação abrem o webui aberto paraqualquerAPI, Modelo ou Automação-não apenas os pontos de extremidade compatíveis com o OpenII.
  • Pense além do LLMS: integrar ferramentas, APIs, scripts locais ou toda a sua casa inteligente.

⚠️ Notas de segurança

  • SempreRevise o código da função antes de ativar.
  • Use apenas plugins de fontes confiáveis.
  • Você tem o poder de aprimorar (ou quebrar!) Seu webui - use com responsabilidade.
